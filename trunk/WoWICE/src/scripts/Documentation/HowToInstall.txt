How to install WoWICE Scripts?
Windows:

1: Compile WoWICE (ULTIMATE REV) with C++ (2008 or last)
2: Compile your WoWICEScripts with C++ (2008 or last)
3: Copy all .LUA files in the LUA Scripting foulder to your script_bin foulder.
4: Import to your world database the WoWICEScripts_Full.sql
5: Set up your server and HAVE FUN!
